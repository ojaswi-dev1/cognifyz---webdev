Responsive webpage with mobile-friendly navigation

This demo converts a basic webpage to responsive layout using CSS media queries. On narrow screens the navigation collapses into a hamburger menu that expands a vertical nav — implemented using a CSS-only checkbox toggle for simplicity and accessibility.

Responsive illustration

Features implemented

  • Grid layout that becomes single-column on mobile
  • Typography and image scaling
  • Accessible CSS-only hamburger menu (no JS)

Sidebar / Project info

Author: Ojaswi Dwivedi
Frameworks: Plain CSS (media queries)
Test breakpoints: <720px mobile, 720–900 tablet, >900 desktop
